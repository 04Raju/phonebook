
# PhoneBook Web Application

PhoneBook Web Application where user can create there account and store there contacts, and access there contact all over the world.



## Tech Stack

**Client:** Html5, Css3, Javascript, Bootstrap 4.6

**Server:** Servlet, Jsp

**Database:** Mysql
